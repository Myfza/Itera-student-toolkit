# 🎓 itera-student-toolkit

## ✅ Tips Verifikasi

- Gunakan email kampus `.ac.id`.
- Siapkan KTM atau surat aktif, beberapa layanan pakai verifikasi via SheerID.
- Periksa eligibility khusus berdasarkan regional/institusi sebelum mendaftar.
